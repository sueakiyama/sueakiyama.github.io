## 0. はじめに

こんにちは、秋山翠花です。まだまだ寒い日が続いていますが、お元気ですか。

私はここ半年、実家と東京を往復する生活を続けていまして、毎週かならず1晩は、東京のホテルに泊まっていました。

そういうわけで、東京でのホテル、特に安ホテルの探し方が若干上手になった (気がする) ので、ブログに書き留めておこうと思います。どうぞお付き合いください。

## 1. 手順

ホテルを探す方法は簡潔。手順は3つしかありません。つまり、

1. 予約サイトで良い感じのホテルを見繕う
2. Google Map のレビューを確認する
3. いちばん安い方法で予約する

この3つです。簡単ですよね。これを書いてしまうと、この記事は終わったも同然なのですが、さすがにそれは寂しすぎるので、各手順について補足説明をしていきます。

## 2. 予約サイト

世は予約サイト大戦国時代。ホテルを予約できるサイトは星の数ほどありますし、「これが一番いい!」と断言できるようなサービスはありません。そういうわけで、この節では、私が利用したことのある予約サイトについて、その長所や短所をご紹介したいと思います。

### 2.1. Booking.com

![booking.comのスクリーンショット](https://sueakiyama.github.io/images/20260225_0.webp)

言わずと知れた業界最大手 (たぶん)、[booking.com](https://booking.com)。ドミトリーから民泊、さらには高級ホテルまで、豊富なジャンルの宿を取り揃えた、まさにホテルのデパートです。長所は何と言っても、その掲載数の多さ。特に、価格帯の安い宿泊施設が多く、便利です。また、会員割引サービス (Genious) のステージが、金額ではなく「宿泊予約回数」で決まるので、安宿にばかり泊まっていても、簡単にステージを上げて、割引を受けることができます (15回予約するとLv.3 (最高ランク) になるのですが、正直、5回の宿泊で上がれる Lv.2 になっておけば十分だと思います。Lv.3 と Lv.2 の割引率の差は大きくありません)。

注意点としては、「必ずスマホからアクセスする」ようにしましょう。どういう意図なのかよく分かりませんが、Booking.com には「モバイル限定料金」という概念があって、スマホブラウザ・アプリから予約したときの料金が、PCブラウザから予約したときより低廉に設定されているのです。なんだか悔しいですが、おとなしくスマホを使いましょう。

### 2.2. じゃらんnet

![じゃらんnetのスクリーンショット](https://sueakiyama.github.io/images/20260225_1.webp)

たぶん日本最大手 (こればっかですね) のホテル予約サイト、[じゃらんnet](https://jalan.net)。じゃらんnetにしか掲載されていない宿も多いので、ときどき使っています。価格帯は booking.com や他の予約サイトに比べて高めですが、定期的に配布されるクーポンを使えば、わりと安く泊まることができます。特に、毎月20日からの「じゃらんのお得な10日間」では、下限金額なしの1000円引きクーポンが配布されるので、ぜひ使いましょう (東京の安宿の相場は2000円～2500円なので、だいたい1000円台で宿泊できます。変だ……)。

以上が、私が主に使う予約サイトです。基本的に、この2つで用は足ります。でもせっかくなので、私が「ごくたまにしか使わない」予約サイトもご紹介したいと思います。

### 2.3. Trip.com

![Trip.comのスクリーンショット](https://sueakiyama.github.io/images/20260225_2.webp)

中国系旅行サイトの雄、[Trip.com][https://trip.com]。体感としては、booking.com より数百円高いホテルが並んでいる気がしますが、ごくたまに、booking より安いホテルが残っていることがあります。長所は細かい会員システム……かな?　いちばん上のランクになると、空港のラウンジが年に12回使えたり、無料で鉄道がキャンセルできたり、eSIM をもらえたりするらしいです。年に2万ドル使わないと上がれないらしいけど…… (誰が Trip.com で年に2万ドルも使うんだ?) まあ私の会員ランクでも、年1回は空港のラウンジに行けるらしいですよ。今度行ってみようかな。

短所は、中国系サイト特有のごちゃごちゃ感 (Yahoo! Japan にちょっと似てるかも) と、アプリの重さ、異常なほどのポップアップ (広告じゃなくて、キャンペーンのお知らせとかではありますが) など。「おれは予約を確認したいだけなのに、なんでポップアップを3個閉じさせられて、謎の通知を見なきゃいけないんだ?」みたいな気分になることがあります。

### 2.4. Airbnb

![Airbnbのスクリーンショット](https://sueakiyama.github.io/images/20260225_3.webp)

民泊の元祖 (本当か?)、[Airbnb](https://airbnb.jp)。正直言うと、安宿探しには向いていません。たまに、何を思ったのか Airb にしか掲載していないドミトリーがあったりして、役に立つことはあります。Airb の最大の欠点は、検索結果のソートができないこと。検索結果を「安い順」に並べ替えがちな私としては、不便さを感じることが多いです。とはいえ、価格帯で絞り込みをかければ、ほとんど同じことですが。

![Airbnb「これまでに訪れた旅行先」のスクリーンショット](https://sueakiyama.github.io/images/20260225_4.webp)

個人的には、Airb の最大の長所って、「これまでに訪れた旅行先」欄じゃないかと思います。マイページに鎮座している「旅行先」を見ると、昔泊まった部屋、遊びに行った場所、会った人のことを思い出して、うれしくなります。なんといっても、パスポート・スタンプ風のデザインがお洒落ですし。

### 2.5. Agoda

![Agodaのスクリーンショット](https://sueakiyama.github.io/images/20260225_5.webp)

何かと (悪い意味で) 話題になりがちな [Agoda](https://agoda.com)。私はまだトラブルに遭ったことはないのですが、行く先々の宿で、「Agodaとは直接契約していません」「Agodaからチェックインコードが送られてこない方は、Agodaに連絡してください。ホテル側では対応できません」などの貼り紙を見かけて、恐れおののいています。

基本的には使わないのですが (怖いので)、ごくたまに、信じられない安さの宿が掲載されていることがあり、「背に腹は代えられぬ……!」と思いながら予約しています。今のところ大丈夫です。

短所は色々あって、Trip.com と同様、デザインがごちゃごちゃしていて、アプリが重い、というのがその一つです。また、「セール」「80％オフ」など、大きな割引がなされているような表示がそこら中にありますが、逆に定価で販売されているところを見たことがありません (定価は他サイトの5倍くらいだと思います)。さらに、アプリと Web で値段が違います。基本的には Web で予約したほうが安い気がします。まあ、他のサイトより明らかに安い、とかでなければ、使わないほうがいいと思います。

### 2.6. Hostelworld

![Hostelworldのスクリーンショット](https://sueakiyama.github.io/images/20260225_6.webp)

[Hostelworld](https://hostelworld.com)は、海外、特にヨーロッパのホステルに強い予約サイトです。逆に、日本の宿の掲載数は少なめ…… (なんでだろう?)

EU・アメリカ・カナダなどの大学に学籍がある方は、[Student Beans](https://studentbeans.com) から7％引きのクーポンが取得できるので、うまく使えば安く泊まれるかもしれません。

## 3. Google Map のレビュー

予約サイトで宿に目星をつけたら、次は Google Map のレビューを確認しましょう。予約サイトのレビューを見てもいいのですが、点数が高く出がちなので……

レビューを見るとき、私が気にしているのは、次の2点です。

1. レビューの平均値。☆3未満 (Google は☆5が満点です) の宿は避けます。こうした宿には、清潔さ、広さ、スタッフの対応などに致命的な問題があることが多いためです。また、東京には掃いて捨てるほど宿泊施設がありますから、わざわざ☆3未満のホテルに泊まる必要もないと思います。
2. 直近のレビュー。特に、トコジラミが発生していないか注意します。レビューの平均値が良くても、直近のレビューに☆1が多い、といったような宿も避けます。

この2点に問題がなければ、あとは予約するだけです。

## 4. 予約

正直、ホテルを見つけたサイトでそのまま予約してしまってもいいと思います。ただ、ときどき損をすることがあるので、次の場所をチェックしておきましょう。

1. 公式サイトの価格。予約サイトの価格は、手数料や掲載料などが上乗せされていて、若干高くなっていることがあります。公式サイトがない場合は諦めましょう。
2. [trivago](https://www.trivago.jp)。正直省略してもいいです。ごくたまに、もっと安い予約方法が見つかります。

## 5. おわりに

ここまで読み切ったあなたは、もうホテル探し上級者です!　いろいろな宿を試してみてくださいね。

おそらく、都内でまともに宿泊できる最安の施設 (1泊2000円) です。ここより安いホテルは、booking のレビューが10点満点で3点だったり、トコジラミが出ていたりします。最安ということもあって、設備は最低限です。シーツはセルフサービスだし、若干衛生面に不安があったりするし (基本的には大丈夫なのですが、いちど前泊まっていた人の強い体臭が残っていたことがあって、大変でした)。あと、シャワーに石鹸がないので、フロントで使い切りパックを買うか、自分で持ち込むかしなければなりません。ところどころ立て付けが悪いところもあって、運悪くドア付近のベッドに当たると、ドアの開閉音で眠れないこともあります。

この施設の強みは、大人数の宿泊を前提にした設備が整っていること。シャワーは (男性用だけで) 10基ほどあるし、トイレも各階に十分揃っています。談話室も十分な広さがあります。また、タオルが無制限に借りられる点、ロッカー (という名前の小部屋) がかなり広い点なども魅力です。私も文句を言いつつ、何度も泊まってしまっています。
